How to use Git?


Every time you install git, you must tell the system who you are.



Every time you work with Git, you pass through four stages:

   1. Creating repository
Go to your projects`s folder and turn it into git project

   2. Checking status

   3. Adding files
Before saving, you must "number them"

And finally - Commit
This is saving the last change in your working




If you don`t wanna write commands , you can use Github dekstop. The work is the same but with buttons



Github for example are used to save your entire
work in the server to not lose or to work with other people.
You publish the branches with the commits there,
push other commits,
share your projects, and so on



Golden rules:

   1. Commit by minutes

   2. Write messages that have sense

   3. Don`t have fear from errors


In Git everything can be turned back to normal
